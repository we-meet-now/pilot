================================================================================
CRITICAL INFORMATION
================================================================================

1. CSS VARIABLES MUST BE FIRST
   - All colors use var(--primary), var(--text), etc.
   - If variables aren't loaded first, styles will be blank

2. ANIMATIONS MUST BE BEFORE COMPONENT FILES
   - Components use @keyframes fadeIn, pulse, float, etc.
   - Must be loaded before components use them

3. KEEP RELATED ITEMS TOGETHER
   - All button variants in one file
   - All avatar variants in one file
   - Don't split by component type

4. TEST EARLY AND OFTEN
   - After each major file group
   - Don't wait until the end

5. PRESERVE SPECIFICITY
   - Some selectors have specific overrides
   - Keep these in the same file as the base selector
